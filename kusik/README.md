## Запуск з нуля

Для запуску лабораторних використовується віртуальне середовище Python, розташоване в корені репозиторію — `.venv` (не зберігається в git).

**1. Створити віртуальне середовище:**
```bash
python -m venv .venv
```

**2. Активувати:**
```bash
# Windows
.venv\Scripts\activate

# Linux / macOS
source .venv/bin/activate
```

**3. Встановити залежності:**
```bash
pip install scikit-learn pandas matplotlib seaborn ipykernel numpy
```

**4. Відкрити ноутбук у VSCode** і вибрати kernel `.venv`.
